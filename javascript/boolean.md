# JavaScript

## Boolean

- Los tipos de datos boolean aceptan un valor del tipo true (verdadero) o false (falso)

**Ejemplo:**

```js
let prendido = true;
let puedeVotar = false;
let casado = false;

console.log(prendido);
console.log(puedeVotar);
console.log(casado);
```
