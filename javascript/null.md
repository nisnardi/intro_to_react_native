# JavaScript

## Null

- Existe un valor llamado **null** para establecer que algo es nulo
- Podemos asignarlo como valor de la siguiente manera:
- También es común tener que validar si un valor es **null**

**Ejemplo:**

```js
let nulo = null;
console.log(nulo);
```
